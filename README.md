# DATA-CLEANING-REPORT-AUTOMATION
The image shows an automated dashboard summarizing the data cleaning process. It highlights key metrics like missing values fixed, duplicates removed, outliers corrected, and standardized entries — all displayed in a clean, professional layout.

📊 Charts Used:

Bar Chart: Displays imputed values for different data fields (e.g., Age, Income, Location).

Table: Lists duplicate records resolved with ID, Name, Date, and Status columns.

Scatter Plot: Shows outlier detection and adjustment results.

Circular Charts (Donut/Pie): Represent overall data quality metrics — completeness, consistency, and accuracy.
